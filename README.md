# BlinkTalk-website
BlinkTalk is a real-time assistive communication system that converts eye blinks into text, speech, and calls/SMS, providing an accessible communication solution for users with severe disabilities.

# 🚀 Features
- Blink Detection: Uses a Raspberry Pi 4 and camera with MediaPipe Face Mesh to track eye blinks via Eye Aspect Ratio (EAR).
- Morse Code Translation: Maps left and right blinks to Morse code for input.
- Text & Speech Output: Converts blinks into text and speech using a TTS module.
- Calls & SMS: Integrates with Twilio for messaging and calls.
- Autocomplete Engine: Speeds up text input for efficient communication.
- # Web Application: Remote configuration of the system, Live system status monitoring, Contact management, Gesture guide for users
  
# 🎯 Purpose
BlinkTalk offers a portable, practical, and highly accessible communication solution for people with severe physical disabilities, allowing them to interact without traditional input devices.

# 🛠️ Technology Stack
- Hardware: Raspberry Pi 4, Camera
- Software: Python, MediaPipe Face Mesh, TTS
- Communication: Twilio API for calls and SMS
- Frontend: Responsive Web Application for system control and monitoring
